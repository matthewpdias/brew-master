## BREWMASTER! :beers:

Get info from a raspi about your homebrew setup.

#### featuring: 

`Raspberry pi`
`Python`
`Flask`

#### Short Term goals:
- Temperature readout remotely
- Alerts for critical temps (too high/too low)
- Graph of temps
